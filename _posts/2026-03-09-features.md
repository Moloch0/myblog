# Repository Features (Linked Doc)

> Linked core doc: `_posts/2026-03-09-config-zh-cn.md`  
> GitHub URL: <https://github.com/Moloch0/myblog/blob/main/_posts/2026-03-09-config-zh-cn.md>

This file describes the current repository features and implementation status. It is not the human core standard. If any conflict exists, follow the core doc.

## 1. Site and Theme

- Stack: `Jekyll` + `jekyll-theme-chirpy`
- Structure: posts in `_posts/`, tab pages in `_tabs/`, shared data in `_data/`
- Permalink: `/posts/:title/`

## 2. Content Features

- Post publishing via Markdown in `_posts/`
- Categories and tags from Front Matter
- Post enhancements: TOC, MathJax, Mermaid, comments enabled by default
- Image zoom via `medium-zoom` (`assets/js/lib/medium-zoom/`)

## 3. Interaction and Analytics

- Comments: `giscus`
- Analytics/page views: `goatcounter`

## 4. Frontend Extension Points

- Custom head include: `_includes/custom-head.html`
- Custom styles: `assets/css/style.css`, `assets/main.scss`
- Custom scripts: `assets/js/custom.js`
- Third-party frontend libs: e.g. `APlayer` (`assets/js/lib/aplayer/`)

## 5. Writing Workflow

- Source of truth for articles: `_posts/`
- Authoring mode: edit/create posts directly in `_posts/*.md`
- Publish mode: normal `git commit` / `git push`

## 6. Dev and Release

- Local run: `bash tools/run.sh`
- Local test: `bash tools/test.sh`
- Deployment: `.github/workflows/pages-deploy.yml` (GitHub Pages)
- PWA is enabled (with cache support)
